# Projeto B7Burguer
O projeto B7 Burguer é um exemplo de desenvolvimento focado na apresentação visual de um menu de lanchonete, utilizando apenas HTML e CSS.
O grande destaque técnico deste projeto é o uso eficiente de Variáveis CSS (custom properties), declaradas no seletor *:root*. Essa abordagem permite:
* Padronização: Armazenar cores, fontes e espaçamentos em um único local centralizado.
* Manutenção: Simplificar futuras alterações de design, que podem ser feitas mudando apenas uma linha no :root em vez de percorrer todo o código.

Além disso, o projeto utiliza técnicas modernas de layout, como o *CSS Grid*, para organizar os cards de hambúrguer de forma responsiva e visualmente agradável, garantindo que o design seja limpo, modular e fácil de escalar. O resultado é uma interface atraente que comunica de forma eficaz os produtos e seus preços, como visto no cardápio de hambúrgueres.
